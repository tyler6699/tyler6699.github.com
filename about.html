Ryan Release & Development Manager & Hobby game developer

Hi my name is Ryan, by day a release and development manager.

I have a degree in Internet Computing and have a background in desktop support, web design, coding and Flash Games.

Have always been passionate about games, starting with an Atari 7800, Amiga, Mega Drive before moving onto PCs and Playstation. First attempted making text games using Easy AMOS and Turbo Pascal, later using Macromedia Flash.

Returned to game development after being inspired by Indie Game the Movie and writing plug-ins for MineCraft servers.

While I enjoy prototyping with LibGDX and have an active project, at the moment am interested in developint 2D JavaScript / HTML games.
